EDUCATION MANAGEMENT INFORMATION SYSTEMS FEATURES

An EMIS is typically a high-level decision-making system used at the government office though it can also be accessed by schools, school authorities and other stakeholders with appropriate permissions.

Systems Configuration and General
Feature Status
A vocabulary system to “localize” terminologies of the country (e.g. Display the Province instead of District, Status of Employment instead of Teacher Registration Status).
A wealth of systems parameters including name of app, country settings, currency, estimated school enrolments parameters, logos, colors, contact details and much more.
A comprehensive access control system to give access to only specific people (or groups of people) to specific features and operations in the system.
Audit trail of what user created/edited what at what time.

Read more about “Configuration through Custom Software Code vs Data”

Modelling the Education System
Feature Status
Highly flexible education system configuration setup without the need to change the code (i.e., configuration through data) stored in the database.
Define up to three different education levels classification alternatives (divides the education system into chronological levels such and Early Childhood Education, Primary Education, Secondary Education).
Configuration of education sectors (functional division of the education system, unlike education levels above which do not overlap, two 2 or more sectors can exist in the same year of education e.g. vocational).
Class level configuration with their ISCED code and sectors.
Support for ISCED Levels essential for international reporting using a common framework.

STUDENT INFORMATION SYSTEM FEATURES

As part of our supported toolkit, we have a Student Information System typically used to improve data management at the school level. This requires more training by more people. It is optional and better suited for schools with an Internet connection and computer literate staff, but it can greatly increase the amount of data collection and improve processes at the school level.

Note that our SIS component is currently undergoing a major modernization of the system to enable the system to work faster even in with low bandwidth environment. A small delay is expected to get all of the classic SIS summarized below if a country chooses to jump to the new shiny SIS directly without waiting. It is expected to be ready as first rough version for this next school year.
